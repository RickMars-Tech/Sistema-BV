# Sistema-BV
Sistema de Biblioteca Virtual (escrito en Python y PyQt).

# Ejecutar 
Para ejecutar desde la Terminal(Linux o Windows), escribe el comando:
```bash
python main_login.py
```
desde la carpeta donde este almazenado, o usa un IDE como PyCharm o VsCodium

# Inicio de Sesion
Para "Iniciar Sesion" por primera ves necesitaras un usuario y una contrase単a, por default agregamos un usuario llamado "admin" cuya contrase単a es "123456" para ser utilizado como usuario principal y crear otros usuarios.
